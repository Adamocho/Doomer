## Supported titles:
- DOOM
- Final DOOM
- DOOM 2
- TNT Evilution
- Plutonia  

**`Incoming support for master levels`**

## Features:
- Select which title you want to play
- Choose dificulty
- Record a demo file
- Join a server
- Host server by yourself
- Choose whether you want:
    - music
    - mouse
    - monsters
    - fast monsters
    - respawning monsters  
    and much, much more...
